Shows a list of most recently added BP activity comments.

== Installation ==
Manually:
1. Upload "last_comments.php" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.

Automatically:
1. Upload "bp_last_comments_plugin.zip" to your plugins directory from Admin Add Plugin page "/wp-admin/plugin-install.php".
2. Activate it.

== Frequently Asked Questions ==
1.  Is the widget shows only single users activity comments or groups too? 
For all types of activity comments.

2. Is there an option to show date added or author? 
Yes, just like in WordPress comments widget.

3. Widget is localized?
Yes. 
There is Russian file is ready, but more languages can be added manually.
